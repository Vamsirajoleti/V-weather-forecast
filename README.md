# V-Weather Forecast

V-Weather Forecast is a modern, premium weather dashboard built with Vanilla JavaScript, HTML5, and CSS3. It fetches real-time data using the free Open-Meteo API and features a beautiful glassmorphism design.

​🌐 Live Demo: https://vamsirajoleti.github.io/V-weather-forecast/

## 🚀 Features
- **Search Any City**: Instantly view weather worldwide.
- **Current Location**: Detects your location automatically via Geolocation API.
- **Dark/Light Mode**: Toggle seamlessly between premium dark and clean light themes.
- **Unit Toggle**: Switch instantly between Celsius and Fahrenheit.
- **24-Hour Forecast**: Scrollable hourly weather updates.
- **7-Day Forecast**: Weekly overview with max and min temperatures.
- **Responsive Design**: Looks amazing on Desktop, Tablet, and Mobile.
- **No API Keys Needed**: Powered by Open-Meteo (No signups required).

## 🛠️ Technology Stack
- **HTML5**: Semantic structure.
- **CSS3**: Custom variables, Flexbox/Grid, Glassmorphism UI, Responsive media queries (No Tailwind/Bootstrap).
- **JavaScript (ES6)**: Async/Await, Fetch API, DOM Manipulation (No React/Libraries).

## 📥 How to Run Locally

1. **Extract the ZIP file** to a folder on your computer.
2. **Open the folder** in your favorite Code Editor (e.g., Visual Studio Code).
3. **Run the Project**:
   - **Method 1 (Best)**: Use the "Live Server" extension in VS Code. Right-click `index.html` and select "Open with Live Server". This ensures the Geolocation feature works perfectly without browser security blocking it.
   - **Method 2 (Quick)**: Double-click `index.html` to open it directly in Chrome, Firefox, or Safari.
